## 프롬프트 A — 30초 자기소개 (엘리베이터 피치)

```
위 데이터를 바탕으로 30초 안에 말할 수 있는 자기소개를 써주세요.

[조건]
- 200자 내외 (말했을 때 30초)
- 상황: {CONTEXT}  (예: 개발자 채용 면접, 네트워킹 행사, SNS 프로필)
- 핵심 메시지: 교육자 출신 개발자, 세종클래스 운영
- 자연스러운 구어체로
```

---

## 프롬프트 B — 풀 자기소개 (면접·소개글)

```
위 데이터를 바탕으로 3~5분 분량의 자기소개를 써주세요.

[조건]
- 상황: {CONTEXT}  (예: 면접, 커뮤니티 소개, 블로그 소개글)
- 구조:
  1. 현재 나는 어떤 사람인가 (1~2문장)
  2. 어떤 여정을 걸어왔는가 (핵심 전환점 3개)
  3. 무엇을 잘하고 무엇을 좋아하는가
  4. 앞으로 무엇을 하고 싶은가
- 방황과 성장의 서사를 솔직하게, 하지만 자신감 있게 표현할 것
- 교육자 → 개발자 전직을 스토리의 핵심으로 활용할 것
```

## 프롬프트 C — 매칭용 프로필 소개 (SNS)

```
위 데이터를 바탕으로 hupository SNS에서 다른 사람들이 이 사람을 만나고 싶어지게 만드는
프로필 소개글을 써주세요.

[조건]
- 400자 내외
- "이런 사람과 이야기 나누고 싶다"는 느낌을 주는 톤
- 관심사·가치관·찾는 연결을 자연스럽게 녹일 것
- 너무 이력서 같지 않게, 사람 냄새 나게
```

# 매칭 설명 생성 프롬프트

## 사용법

두 사람의 데이터를 컨텍스트로 제공한 뒤 이 프롬프트를 실행하세요:
- `person_A/index.yaml` + `person_A/07_matching/`
- `person_B/index.yaml` + `person_B/07_matching/`

---

## 프롬프트 A — 두 사람의 매칭 설명

```
위에 제공된 두 사람의 데이터를 바탕으로
왜 이 두 사람이 잘 맞는지 설명해주세요.

[출력 형식]
- 공통점 3가지 (관심사, 가치관, 목표 중심)
- 서로 보완하는 점 2가지
- 추천 대화 주제 3가지
- 어울리는 만남 형식 (커피챗, 스터디, 프로젝트 등)

[톤] 따뜻하고 구체적으로. 데이팅앱 느낌이 아닌 커뮤니티 연결 느낌으로.
```

---

## 프롬프트 B — 이 사람과 잘 맞는 사람 설명

```
parksejong의 매칭 데이터를 바탕으로
이 사람과 잘 맞을 사람을 구체적으로 묘사해주세요.

[출력]
- 어떤 관심사를 가진 사람인가
- 어떤 삶의 방식을 가진 사람인가
- 어떤 목표를 가진 사람인가
- 구체적인 페르소나 예시 1개 (가상)

[목적] 매칭 알고리즘의 검색 쿼리 생성 또는 사용자에게 보여줄 매칭 설명용
```

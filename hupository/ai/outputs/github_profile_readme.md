**"어떻게 사는 것이 사람답게 사는 것일까?"**

그 질문이 저를 청년 커뮤니티 기획자로, 12개국 국제캠프 PM으로,
수학·입시 교육자로, 그리고 2023년 — 37세에 — 개발자로 이끌었습니다.

지금은 그 모든 경험을 하나로 합쳐, 교육 플랫폼 **세종클래스**를 혼자 만들어 운영하고 있습니다.

## 코드 바깥의 저

뇌과학 책을 180권 넘게 읽었습니다.
의식이 어떻게 생겨나는지, 감정이 몸에서 어떻게 만들어지는지가 궁금해서요.
안토니오 다마지오의 책은 전작을 탐독했고, 조지프 르두와 유발 하라리도 마찬가지입니다.

물리학도 독학으로 대학 교재 수준까지 팠고,
박문호의 자연과학세상(박자세) 정회원으로 활동하고 있습니다.

코드를 잘 짜고 싶은 이유도 결국 같습니다.
좋은 시스템을 만들어 사람의 성장을 돕고 싶어서입니다.
